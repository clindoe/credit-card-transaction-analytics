# Tableau Dashboards

## Live Dashboards (click to interact)

### Dashboard 1: Fraud Overview
**Features:** KPI cards, monthly fraud trend, category fraud rates, hour×day heatmap, geographic map
**Tableau techniques:** Calculated fields, dashboard actions, cross-filtering

### Dashboard 2: Customer Risk Segmentation
**Features:** Risk scatter plot, tier distribution, Top N detail table, risk score histogram
**Tableau techniques:** LOD expressions (FIXED), parameters, dynamic Top N filter, bins

### Dashboard 3: Merchant Category Analysis
**Features:** Category treemap, fraud vs legitimate stacked bar, spending trends, merchant bubbles
**Tableau techniques:** Parameters for date granularity toggle, treemap, dual-axis potential
